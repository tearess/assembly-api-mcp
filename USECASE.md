# 국회 MCP 서버 활용 사례 100선

> assembly-api-mcp를 활용한 실용적인 질문 100가지.
> AI 어시스턴트(Claude, ChatGPT 등)에서 바로 사용할 수 있는 자연어 질문입니다.
> 단순 조회부터 여러 도구를 조합한 복합 분석까지 다양한 난이도를 포함합니다.

## 도구 범례

| 도구 | 모드 | 설명 |
|------|------|------|
| `search_members` | Lite/Full | 의원 검색 (이름/정당/선거구/위원회) |
| `search_bills` | Lite/Full | 의안 검색 (의안명/제안자/상태/상세) |
| `get_schedule` | Lite/Full | 국회 일정 조회 (날짜/위원회/키워드) |
| `search_meetings` | Lite/Full | 회의록 검색 (본회의/위원회/소위/국감/청문회) |
| `get_votes` | Lite/Full | 표결 조회 (전체/의안별 상세) |
| `analyze_legislator` | Lite/Full | 의원 종합분석 (인적사항+발의법안+표결) |
| `track_legislation` | Lite/Full | 주제별 법안 추적 (키워드+심사현황) |
| `discover_apis` | Lite/Full | 276개 API 카탈로그 탐색 |
