## API 공통 사항

### 인증

| 항목 | 내용 |
|------|------|
| 인증 파라미터 | `KEY` 또는 `authKey` |
| 테스트 키 | `sample` (최대 10건) |
| 발급처 | https://open.assembly.go.kr (회원가입 필요) |
| Rate Limit | 개발계정 10,000 요청/월 |

### Base URL

```
https://open.assembly.go.kr/portal/openapi/{API_CODE}
```

### 공통 파라미터

| 파라미터 | 설명 | 필수 | 기본값 |
|----------|------|------|--------|
| `KEY` | API 인증키 | O | - |
| `Type` | 응답 포맷 (`json` / `xml`) | X | `xml` |
| `pIndex` | 페이지 번호 | X | `1` |
| `pSize` | 페이지당 건수 | X | `10` |

### 에러코드

| 코드 | 설명 |
|------|------|
| `INFO-000` | 정상 |
| `INFO-100` | 인증키 미입력 |
| `INFO-200` | 유효하지 않은 인증키 |
| `INFO-300` | 요청제한 초과 |
| `ERROR-300` | 필수 파라미터 누락 |
| `ERROR-331` | 유효하지 않은 파라미터 타입 |
| `ERROR-332` | 유효하지 않은 파라미터 값 |
| `ERROR-333` | 조회 범위 초과 |
| `ERROR-500` | 서버 오류 |
| `ERROR-600` | 데이터베이스 오류 |
| `ERROR-601` | SQL 오류 |
| `DATA-000` | 데이터 없음 |
| `DATA-001` | 데이터 한계 초과 |
