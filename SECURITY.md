# Security Policy

## API 키 관리

이 프로젝트는 [열린국회정보](https://open.assembly.go.kr) API 키를 사용합니다.

- API 키는 반드시 **환경변수**(`ASSEMBLY_API_KEY`)로 관리하세요
- `.env` 파일은 `.gitignore`에 등록되어 있습니다
- 코드에 API 키를 **절대 하드코딩하지 마세요**
- API 키가 노출된 경우 즉시 열린국회정보에서 재발급하세요

## 이 프로젝트의 보안 특성

- **읽기 전용**: 국회 공공 데이터 조회만 수행 (쓰기/수정 없음)
- **SQL 미사용**: REST API만 호출, SQL Injection 해당 없음
- **입력 검증**: 모든 파라미터를 Zod 스키마로 검증
- **URL 인코딩**: `encodeURIComponent()`로 파라미터 인코딩
- **Rate Limit**: 월 10,000건 추적, 80% 임계치 경고
